# SSH-Fingerprint

To run this in DOSBox using masm to compile (https://winworldpc.com/product/macro-assembler/5x):

    > mount c c:\
    > c:
    > cd c:\Users\...  <- place where is masm and your asembler project
    > masm project.asm
    > link project.obj
    > project.exe 1 d5293e9a8d90265d6b6bfb8abba5da23
